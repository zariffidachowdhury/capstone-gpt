# Mock Capstone Projects

## 1. Smart Campus Room Finder
**Description:** A student team builds a web app that helps users find available study rooms and campus resources in real time.

**Sample questions:**
- How should we break this project into sprint-sized backlog items?
- What should go into our project charter for a campus scheduling tool?
- How do we write an executive summary for this idea?

## 2. Peer Tutoring Match Platform
**Description:** A matchmaking system that pairs students with peer tutors based on course needs, availability, and learning goals.

**Sample questions:**
- What GitLab workflow should our tutoring team follow each sprint?
- Which ABET outcomes connect best to this project?
- How should we run a sprint retrospective after a missed milestone?

## 3. Accessible Dining Menu Assistant
**Description:** A tool that helps students filter campus dining options by allergens, dietary needs, and accessibility preferences.

**Sample questions:**
- What stakeholders should we mention in our requirements document?
- How do we justify this project’s value in a proposal?
- What are good acceptance criteria for a dietary filter feature?

## 4. Internship Reflection Dashboard
**Description:** A dashboard that helps students organize internship reflections, outcomes, and skills evidence for academic review.

**Sample questions:**
- How can we map our features to professionalism and documentation expectations?
- What should we discuss in sprint planning for this dashboard?
- How do we present measurable success criteria for this project?

## 5. Senior Design Team Health Tracker
**Description:** A collaboration tool that helps capstone teams monitor blockers, task ownership, and meeting follow-ups across the semester.

**Sample questions:**
- How should we define “done” for collaboration features?
- What risks should we surface in a weekly status update?
- How can Agile ceremonies keep this project on track?
